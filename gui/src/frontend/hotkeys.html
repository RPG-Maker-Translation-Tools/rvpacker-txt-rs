Горячие клавиши
Цифры от 1 до + - Редактирование разных файлов

Tab - Открыть панель выбора файлов

Ctrl + S - Сохранить файлы перевода

Ctrl + F - Навестить на строку поиска

R - Открыть панель поиска

Alt + Enter - Перепрыгнуть на нижнее текстовое поле

Ctrl + Enter - Перепрыгнуть на верхнее текстовое поле

Esc - Если курсор сфокусирован на текстовом поле, убрать фокусировку на текстовом поле; иначе, выйти из состояния редактирования файлов

Alt + C - Скомпилировать в JSON файлы

ЛКМ на результате поиска - перейти к найденному элементу

ПКМ на результате поиска - заменить текст этого элемента на тот, что в данный момент введён в поле замены

Ctrl + G - открыть поле для ввода числа линии, на которую вы хотите перейти
